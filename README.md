# 🔐 Project: Sign up Form.

Proyecto frontend desarrollado con **HTML, CSS y JavaScript puro**, que implementa la creacion de formularios y validación de contraseñas en tiempo real con feedback visual para el usuario.

---

## 📌 Descripción

Este proyecto consiste en un formulario de creación de cuenta que incluye validación dinámica de contraseñas para mejorar la experiencia del usuario.

El sistema:

- Verifica que ambas contraseñas coincidan.
- Muestra mensajes visuales de éxito o error.
- Cambia estilos dinámicamente (bordes rojo / verde).
- Habilita o deshabilita el botón de envío automáticamente.
- Ejecuta validación al salir del campo o mientras se escribe.

---

## 🚀 Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Manipulación del DOM

---

## 🎯 Funcionalidades

✔ Validación en tiempo real  
✔ Comparación de contraseña y confirmación  
✔ Feedback visual dinámico  
✔ Control del botón de envío  
✔ Código modular y separado (HTML / CSS / JS)
